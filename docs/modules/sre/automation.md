# SRE - Automation with Ansible

Colección de playbooks de **Ansible** y scripts de **Python** para la gestión de configuración y remediación automática de incidentes.

## Casos de Uso

1. **Security Hardening:** Aplicación automática de parches de seguridad (CIS Benchmarks).
2. **Disaster Recovery:** Scripts para la restauración automatizada de bases de datos.
