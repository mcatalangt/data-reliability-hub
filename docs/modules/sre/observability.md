# SRE - Observability

Implementación de pila de observabilidad completa utilizando herramientas Open Source.

## Componentes

* **Métricas:** Prometheus + Grafana Mimir
* **Logs:** Grafana Loki
* **Traza (Traces):** Grafana Tempo
* **Recolección:** OpenTelemetry Collector
